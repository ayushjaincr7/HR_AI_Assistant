## 🧠 How It Works

1. User uploads or pastes resume  
2. User enters job description  
3. Prompt Template formats input  
4. OpenAI LLM processes the data  
5. System returns:
   - Strengths
   - Matching skills
   - Final decision (Shortlist / Reject)
